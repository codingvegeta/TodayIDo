application 기본 객체

특정 웹 어플리케이션에 포함된 모든 JSP 페이지는 하나의 application 기본 객체를 공유한다.

-웹 어플리케이션 초기화 파라미터 읽어오기
웹 어플리케이션에서 사용할 수 있는 초기화 파라미터는 WEB-INF\web.xml파일에 <context-para>태그를 사용하여 추가한다.


-서버 정보 읽어오기
application 기본 객체는 현재 사용중인 웹 컨테이너에 대한 정보를 읽어오는 메서드를 제공하고있다.
getServerInfo() : 서버 정보를 구한다.
getMajorVersion() : 서버가 지원하는 서블릿 규약의 메이저 버전을 리턴한다.
getMinorVersion() : 서버가 지원하는 서블릿 규약의 마이너 버전을 리턴한다.

-로그 메시지 기록하기
log 기능을 사용해서 로그를 메시지를 기록 할 수 있다.

-웹 어플리케이션의 자원 구하기
JSP 페이지에서 웹 어플리케이션 폴더에 위치한 파일을 사용해야 할 때가 있다.
이 경우 절대 경로를 사용하여 지정한 자원을 읽어올 수 있다.
