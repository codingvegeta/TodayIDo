스크립트 요소

* 스크립트릿
* 표현식
* 선언부

-스크립트릿
JSP 페이지에서 자바 코드를 실행할 때 사용하는 코드 블록
<% 로 시작(실행할 자바 코드를 적는다) %>로 종료
oneToTen.jsp , sum.jsp , 를 확인해보자

-표현식
어떤 값을 출력 결과에 포함시키고자 할 때 사용
<%= 값 %> 계산해서 값을 생성할 수 도 있다.
oneToTen2.jsp를 실행시켜 보자.

-선언부
JSP 페이지의 스크립트릿이나 표현식에서 사용할 수 있는 메서드를 작성할 떄에는 선언부를 사용한다.
<%! %> 선언부의 함수는 자바의 메서드와 동일하며, 각 요소는 다음과 같은 의미를 지닌다.
* 리턴타입 : 메서드 실행 결과값의 타입을 지정한다.
* 메서드 이름: 메서드의 이름을 의미한다.
* 파라미터 목록 : 콤마로 구분된 파라미터 목록을 지정한다.(메서드 내에서 사용할 변수)
* 자바코드1-n : 메서드 내에서 실행할 자바 코드가 위치한다.
* return 값 : 메서드의 실행 결과로 값을 리턴

useDecle.jsp를 실행시켜 보자.
스크립트릿 에서도 선언부에 정의한 메서드를 사용할 수 있다.
useDecle2.jsp를 실행시켜 보자.
